# 💼 Job Application Tracker

A modern and responsive Job Application Tracker that helps users manage job applications efficiently. Users can track application status (Interview / Rejected), view job details, and manage their progress from a clean dashboard UI.

🔗 Live Site: https://your-live-link.com  
🔗 GitHub Repo: https://github.com/your-username/job-application-tracker  

---

## 📌 Project Overview

This project is a dashboard-based job tracking system where users can:
- View available jobs
- Mark jobs as Interview or Rejected
- Track job status dynamically
- Manage job applications visually

The UI is fully responsive and works smoothly on both desktop and mobile devices.

## 🚀 Technologies Used

- React.js

## ✨ Key Features

- 📊 Dashboard with job status count
- 🧾 Job cards with detailed information
- 🔄 Toggle between Interview and Rejected status
- 📂 Tab system (All / Interview / Rejected)
- ❌ Delete job functionality
- 📱 Fully responsive design
- 🚫 No dummy (lorem ipsum) content

## How to Run Locally

### 1. Clone the repository
git clone https://github.com/your-username/job-application-tracker.git

### 2. Go to project folder
cd job-application-tracker

### 3. Install dependencies
npm install

### 4. Run the project
npm run dev

Then open:
👉 http://localhost:5173
