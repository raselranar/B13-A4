# 💼 Job Application Tracker

A modern and responsive Job Application Tracker that helps users manage job applications efficiently. Users can track application status (Interview / Rejected), view job details, and manage their progress from a clean dashboard UI.

🔗 Live Site: https://raselranar.github.io/B13-A4/

🔗 GitHub Repo: https://github.com/raselranar/B13-A4

---

## 📌 Project Overview

This project is a dashboard-based job tracking system where users can:
- View available jobs
- Mark jobs as Interview or Rejected
- Track job status dynamically
- Manage job applications visually

The UI is fully responsive and works smoothly on both desktop and mobile devices.

## How to Run Locally

### 1. Clone the repository
git clone https://github.com/raselranar/B13-A4.git

### 2. Go to project folder
cd job-application-tracker

### 3. Install dependencies
npm install

### 4. Run the project
npm run dev

Then open:
👉 http://localhost:5173
---
## Functionalities

- Clicking **Interview**
    
    - Adds job to Interview tab
        
    - Updates dashboard count
        
- Clicking **Rejected**
    
    - Adds job to Rejected tab
        
    - Updates dashboard count
        
- Toggle Feature
    
    - Switch between Interview ↔ Rejected
        
    - Updates UI and counts instantly
        
- Delete Feature
    
    - Removes job card
        
    - Updates dashboard + tab counts
